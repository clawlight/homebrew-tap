# clawlight Homebrew tap

Homebrew formulae for [clawlight](https://github.com/clawlight/clawlight-cli) —
a TUI dashboard and macOS menu bar indicator for monitoring Claude Code sessions.

## Install

```bash
brew install clawlight/tap/clawlight
```

## How it's published

`Formula/clawlight.rb` is generated and pushed automatically by the
[release workflow](https://github.com/clawlight/clawlight-cli/blob/main/.github/workflows/release.yml)
in the `clawlight-cli` repo on each tagged `vX.Y.Z` release. You normally
don't edit it by hand.
